## Требование CEO (дословно)

Выгрузить проект на GitHub, задеплоить на Netlify, получить публичную ссылку (как на netlify.com), чтобы CEO мог отправить ссылку заказчику для просмотра без localhost.

## Шаги выполнения

1. Сбилдить `apps/landing/` в production (`pnpm build` -> `dist/`)
2. Закоммитить все файлы проекта в git
3. Запушить на GitHub (remote origin)
4. Задеплоить `apps/landing/dist/` на Netlify через CLI
5. Получить публичную ссылку вида `https://xxx.netlify.app`
6. Предоставить ссылку CEO
