# CEO PROMPTS — Набор промптов для вставки в задачу

> **Как использовать:** Копируй нужный промпт + пиши свою задачу. Вставляй в чат Claude Code.
> Без промпта агент ПРОЕБЁТСЯ. С промптом — выполнит по шагам.

ЗАДАЧА: [твоя задача здесь]

## ПАМЯТКА

| Промпт | Когда использовать |
|--------|-------------------|
| **P1** | Любая стандартная задача |
| **P2** | Что-то сломалось, баг |
| **P3** | Новая функциональность |
| **P4** | Визуал, CSS, дизайн, UI |
| **P5** | Деплой, DNS, хостинг, CI/CD |

| **P7** | Запуск нового проекта с нуля |
| **P8** | Мелочь: цвет кнопки, текст, иконка |
| **P9** | Несколько задач параллельно |
