# EDMI — Мобільний додаток для стоматологічних мікроскопів

## Презентація продукту

---

## Про проєкт

**EDMI** — повнофункціональний мобільний додаток для компанії EDMI (edmi.com.ua), провідного постачальника стоматологічних мікроскопів та оптичного обладнання в Україні та Європі.

Додаток надає стоматологам прямий доступ до каталогу обладнання, інтерактивного конфігуратора мікроскопів, AI-асистента та AR-візуалізації — просто зі смартфона.

---

## Функціонал додатку

### 1. Каталог продукції

Повний каталог стоматологічних мікроскопів із **синхронізацією в реальному часі** з основним сайтом edmi.com.ua через WooCommerce API.

- Головний екран у стилі провідних маркетплейсів (Rozetka-style)
- Hero-секція з градієнтним фоном та пошуком
- Quick Action кнопки для швидкого доступу до функцій
- Рекомендації та найкращі пропозиції
- Сітка категорій з реальними зображеннями
- Фільтрація за брендом та сортування
- Картки товарів з цінами, знижками, статусами наявності

### 2. Сторінка товару (PDP)

Детальна презентація кожного мікроскопа:

- Галерея фотографій з пагінацією та свайпом
- Бейджі: новий / б/в / знижка
- Характеристики в розгортних акордеонах
- Інформація про доставку та оплату
- Кнопка B2B «Запросити рахунок-фактуру»
- Пов'язані товари
- Нижня панель: порівняння, кошик, обране, «Купити»

### 3. Конфігуратор мікроскопа

Унікальна функція — **інтерактивне складання мікроскопа** під потреби клієнта:

- Вибір з 4 моделей: Zeiss Pico Mora, Zeiss PROergo, Leica M320, CJ-Optik Flexion
- 5 груп опцій: колір, кріплення, освітлення, об'єктив, камера
- Реальні ціни-дельти в євро для кожної опції
- Калькуляція вартості в реальному часі
- Карусель фотографій обраної моделі
- Додавання конфігурації в кошик одним натисканням

### 4. EDMik — AI-асистент

Вбудований чат-бот **EDMik** доступний на кожному екрані:

- Floating Action Button (фірмовий фіолетовий колір EDMI)
- Розпізнавання намірів за 73 ключовими словами
- 8 типів відповідей: рекомендації, навігація, картки товарів
- Quick Actions: «Повторити замовлення», «Статус замовлення», «Зв'язатися з менеджером»
- Картки товарів прямо в чаті з переходом на PDP
- Підготовлений для підключення Claude AI та голосового введення

### 5. AR-візуалізація «Переглянути в кабінеті»

Технологія доповненої реальності для примірки мікроскопа:

- Повноекранний AR-режим з камерою пристрою
- Автоматичне виявлення горизонтальних поверхонь
- Розміщення 3D-моделі мікроскопа одним дотиком
- Жести: переміщення, обертання двома пальцями, масштабування
- Скидання позиції та повторне розміщення
- Fallback-екран для пристроїв без AR-підтримки

### 6. Кошик та Checkout

Повноцінний процес покупки:

- Кошик з керуванням кількістю (+/-), видаленням, очищенням
- 4-кроковий checkout: Контакти → Доставка → Оплата → Підтвердження
- Підтримка LiqPay та WayForPay (підготовлена інтеграція)
- Інтеграція з Новою Поштою для доставки
- Підсумкова сума з розбивкою

### 7. Профіль та замовлення

- Авторизація (телефон / email)
- Історія замовлень з фільтрацією за статусом
- Деталі замовлення з timeline відстеження
- Обрані товари
- Перемикання мови (UK / EN)

## Технічна архітектура

### Стек технологій

| Шар | Технологія |
|-----|-----------|
| **Mobile** | React Native + Expo SDK 54 (New Architecture) |
| **Навігація** | expo-router v6 (файлова маршрутизація) |
| **Стан** | Zustand v5 (7 stores) + TanStack Query v5 |
| **UI** | Кастомна дизайн-система (37 компонентів) |
| **3D / AR** | @reactvision/react-viro (ViroReact) |
| **i18n** | i18next + react-i18next (українська, англійська) |
| **Backend** | Node.js + Express + PostgreSQL + Redis |
| **API** | WooCommerce Store API (публічний, реальні дані) |
| **AI** | Anthropic Claude API (підготовлено) |
| **Типізація** | TypeScript (strict mode) |

### Архітектура проєкту

```
edmi/
├── apps/
│   ├── mobile/          ← React Native (Expo)
│   │   ├── app/         ← 18 маршрутів (file-based routing)
│   │   ├── components/  ← 37 React-компонентів
│   │   ├── stores/      ← 7 Zustand stores
│   │   ├── hooks/       ← TanStack Query hooks
│   │   └── screens/     ← Lazy-loaded екрани (AR)
│   └── admin/           ← Admin-панель (Vite + React)
├── packages/
│   ├── shared/          ← 24 модулі (типи, валідатори, моки, i18n)
│   └── server/          ← Express middleware сервер
└── docs/                ← Документація (6 файлів)
```

### Ключові архітектурні рішення

1. **Монорепо (pnpm workspaces)** — єдина кодова база для mobile, server, admin та shared-пакету
2. **WooCommerce як джерело істини** — реальні дані з сайту, жодних дублікатів
3. **Shared-пакет** — типи, валідатори та переклади використовуються і на клієнті, і на сервері
4. **Offline-capable** — TanStack Query + AsyncStorage для кешування каталогу
5. **Lazy loading для AR** — ViroReact завантажується лише при переході на AR-екран
6. **Серверні підписи** — всі криптографічні операції лише на сервері

## Методологія розробки

### AI-driven архітектура

Проєкт реалізований методом **AI-augmented development** — де архітектор (людина) визначає бачення, а AI-асистент (Claude) виконує реалізацію.

**Як це працювало:**

1. **Стратегічне планування** — розробник визначав фази, пріоритети та архітектурні рішення
2. **Дослідження та аналіз** — AI-агенти паралельно досліджували кодову базу, API документацію та залежності
3. **Структурне проєктування** — план кожної фази затверджувався до початку реалізації
4. **Ітеративна реалізація** — пофазне створення з тестуванням на реальному пристрої після кожного етапу
5. **Миттєве налагодження** — баги аналізувалися агентами з доказовою базою з вихідного коду

**Чому це працює:**

- Архітектор фокусується на **що** і **навіщо**, AI — на **як**
- Кожне рішення документується та обґрунтовується
- Паралельне дослідження: 3 AI-агенти одночасно аналізують різні аспекти задачі
- Нульовий технічний борг — strict TypeScript, валідація на кожному кроці
- Повна документація проєкту створюється в процесі розробки

## Унікальні переваги

### Для бізнесу EDMI

- **Прямий канал продажів** — минаючи маркетплейси
- **AR-демонстрація** — клієнт бачить мікроскоп у своєму кабінеті до покупки
- **Конфігуратор** — збільшує середній чек за рахунок up-sell опцій
- **AI-асистент** — знижує навантаження на менеджерів
- **Offline-каталог** — працює навіть без інтернету

### Для стоматологів

- **Професійний інструмент** — не просто магазин, а помічник у виборі обладнання
- **AR примірка** — об'єктивна оцінка розмірів мікроскопа в реальному просторі
- **B2B функції** — запит рахунку-фактури, історія замовлень
- **Мультимовність** — українська та англійська мови інтерфейсу
