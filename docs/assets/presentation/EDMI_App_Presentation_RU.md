# EDMI — Мобильное приложение для стоматологических микроскопов

## Презентация продукта

---

## О проекте

**EDMI** — полнофункциональное мобильное приложение для компании EDMI (edmi.com.ua), ведущего поставщика стоматологических микроскопов и оптического оборудования в Украине и Европе.

Приложение предоставляет стоматологам прямой доступ к каталогу оборудования, интерактивному конфигуратору микроскопов, AI-ассистенту и AR-визуализации — прямо со смартфона.

---

## Функционал приложения

### 1. Каталог продукции

Полный каталог стоматологических микроскопов с **синхронизацией в реальном времени** с основным сайтом edmi.com.ua через WooCommerce API.

- Главный экран в стиле ведущих маркетплейсов (Rozetka-style)
- Hero-секция с градиентным фоном и поиском
- Quick Action кнопки для быстрого доступа к функциям
- Рекомендации и лучшие предложения
- Сетка категорий с реальными изображениями
- Фильтрация по бренду и сортировка
- Карточки товаров с ценами, скидками, статусами наличия

### 2. Страница товара (PDP)

Детальная презентация каждого микроскопа:

- Галерея фотографий с пагинацией и свайпом
- Бейджи: новый / б/у / скидка
- Характеристики в раскрывающихся аккордеонах
- Информация о доставке и оплате
- Кнопка B2B "Запросить счет-фактуру"
- Связанные товары
- Нижняя панель: сравнение, корзина, избранное, "Купить"

### 3. Конфигуратор микроскопа

Уникальная функция — **интерактивная сборка микроскопа** под нужды клиента:

- Выбор из 4 моделей: Zeiss Pico Mora, Zeiss PROergo, Leica M320, CJ-Optik Flexion
- 5 групп опций: цвет, крепление, освещение, объектив, камера
- Реальные цены-дельты в евро для каждой опции
- Калькуляция стоимости в реальном времени
- Карусель фотографий выбранной модели
- Добавление конфигурации в корзину одним нажатием

### 4. EDMik — AI-ассистент

Встроенный чат-бот **EDMik** доступен на каждом экране:

- Floating Action Button (фирменный фиолетовый цвет EDMI)
- Распознавание намерений по 73 ключевым словам
- 8 типов ответов: рекомендации, навигация, карточки товаров
- Quick Actions: "Повторить заказ", "Статус заказа", "Связаться с менеджером"
- Карточки товаров прямо в чате с переходом на PDP
- Подготовлен для подключения Claude AI и голосового ввода

### 5. AR-визуализация "Посмотреть в кабинете"

Технология дополненной реальности для примерки микроскопа:

- Полноэкранный AR-режим с камерой устройства
- Автоматическое обнаружение горизонтальных поверхностей
- Размещение 3D-модели микроскопа одним касанием
- Жесты: перемещение, вращение двумя пальцами, масштабирование
- Сброс позиции и повторное размещение
- Fallback-экран для устройств без AR-поддержки

### 6. Корзина и Checkout

Полноценный процесс покупки:

- Корзина с управлением количеством (+/-), удалением, очисткой
- 4-шаговый checkout: Контакты → Доставка → Оплата → Подтверждение
- Поддержка LiqPay и WayForPay (подготовлена интеграция)
- Интеграция с Новой Поштой для доставки
- Итоговая сумма с разбивкой

### 7. Профиль и заказы

- Авторизация (телефон / email)
- История заказов с фильтрацией по статусу
- Детали заказа с timeline отслеживания
- Избранные товары
- Переключение языка (UK / EN)

## Техническая архитектура

### Стек технологий

| Слой | Технология |
|------|-----------|
| **Mobile** | React Native + Expo SDK 54 (New Architecture) |
| **Навигация** | expo-router v6 (файловая маршрутизация) |
| **Состояние** | Zustand v5 (7 stores) + TanStack Query v5 |
| **UI** | Кастомная дизайн-система (37 компонентов) |
| **3D / AR** | @reactvision/react-viro (ViroReact) |
| **i18n** | i18next + react-i18next (украинский, английский) |
| **Backend** | Node.js + Express + PostgreSQL + Redis |
| **API** | WooCommerce Store API (публичный, реальные данные) |
| **AI** | Anthropic Claude API (подготовлено) |
| **Типизация** | TypeScript (strict mode) |

### Архитектура проекта

```
edmi/
├── apps/
│   ├── mobile/          ← React Native (Expo)
│   │   ├── app/         ← 18 маршрутов (file-based routing)
│   │   ├── components/  ← 37 React-компонентов
│   │   ├── stores/      ← 7 Zustand stores
│   │   ├── hooks/       ← TanStack Query hooks
│   │   └── screens/     ← Lazy-loaded экраны (AR)
│   └── admin/           ← Admin-панель (Vite + React)
├── packages/
│   ├── shared/          ← 24 модуля (типы, валидаторы, моки, i18n)
│   └── server/          ← Express middleware сервер
└── docs/                ← Документация (6 файлов)
```

### Ключевые архитектурные решения

1. **Монорепо (pnpm workspaces)** — единая кодовая база для mobile, server, admin и shared-пакета
2. **WooCommerce как источник истины** — реальные данные с сайта, никаких дубликатов
3. **Shared-пакет** — типы, валидаторы и переводы используются и на клиенте, и на сервере
4. **Offline-capable** — TanStack Query + AsyncStorage для кеширования каталога
5. **Lazy loading для AR** — ViroReact загружается только при переходе на AR-экран
6. **Серверные подписи** — все криптографические операции только на сервере

## Методология разработки

### AI-driven архитектура

Проект реализован методом **AI-augmented development** — где архитектор (человек) определяет видение, а AI-ассистент (Claude) выполняет реализацию.

**Как это работало:**

1. **Стратегическое планирование** — разработчик определял фазы, приоритеты и архитектурные решения
2. **Исследование и анализ** — AI-агенты параллельно исследовали кодовую базу, API документацию и зависимости
3. **Структурное проектирование** — план каждой фазы утверждался до начала реализации
4. **Итеративная реализация** — пофазное создание с тестированием на реальном устройстве после каждого этапа
5. **Мгновенная отладка** — баги анализировались агентами с доказательной базой из исходного кода

**Почему это работает:**

- Архитектор фокусируется на **что** и **зачем**, AI — на **как**
- Каждое решение документируется и обосновывается
- Параллельное исследование: 3 AI-агента одновременно анализируют разные аспекты задачи
- Нулевой технический долг — strict TypeScript, валидация на каждом шаге
- Полная документация проекта создается в процессе разработки

## Уникальные преимущества

### Для бизнеса EDMI

- **Прямой канал продаж** — минуя маркетплейсы
- **AR-демонстрация** — клиент видит микроскоп в своем кабинете до покупки
- **Конфигуратор** — увеличивает средний чек за счет up-sell опций
- **AI-ассистент** — снижает нагрузку на менеджеров
- **Offline-каталог** — работает даже без интернета

### Для стоматологов

- **Профессиональный инструмент** — не просто магазин, а помощник в выборе оборудования
- **AR примерка** — объективная оценка размеров микроскопа в реальном пространстве
- **B2B функции** — запрос счета-фактуры, история заказов
- **Мультиязычность** — украинский и английский интерфейс
